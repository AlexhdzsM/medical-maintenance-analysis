# Medical Maintenance Analysis

## Objective
Analyze medical equipment maintenance data to identify failure patterns, optimize response times, and automate technical reports.

## Problem
Hospitals rely on reactive maintenance, leading to increased downtime and inefficiencies. This project aims to use data to improve decision-making.

## Tools
- Python
- Pandas
- (Power BI / Matplotlib próximamente)

## Project Status
In progress 🚧

## Key Findings

- X-Ray equipment generates the highest downtime despite fewer occurrences.
- Maintenance efforts may need to prioritize high-impact equipment rather than frequent failures.
